Privacy Policy

Effective Date: August 2025

Last Updated: August 2025

Summary: We process translation requests in real-time without storing your content. Your privacy is our priority.

1. Information We Collect

1.1 Translation Data

  • Text messages you submit for translation
  • Detected source language
  • Target language for translation

1.2 Technical Data

  • ServiceM8 authentication tokens (temporary)
  • API request logs (no message content)
  • Error logs for service improvement

2. How We Use Your Information

  • Translation Processing: Your text is sent to OpenAI's API for translation
  • Service Delivery: Returning translated content to your ServiceM8 interface
  • Quality Assurance: Monitoring API performance (no content stored)
  • Support: Troubleshooting technical issues when contacted

3. Data Storage and Retention

Important: We do NOT store your translated messages or content.
  • Translation Content: Processed in real-time, immediately discarded
  • Authentication Tokens: Stored temporarily for session duration only
  • API Logs: Basic request metadata retained for 30 days (no message content)
  • Error Logs: Technical data only, retained for 90 days

4. Third-Party Services

4.1 OpenAI

  • Your translation requests are processed through OpenAI's API
  • OpenAI's data usage policy applies to translation processing
  • As of our knowledge, OpenAI does not store API requests for GPT-4
  • View OpenAI's privacy policy: https://openai.com/privacy/

4.2 ServiceM8

  • Integration requires ServiceM8 OAuth authentication
  • We access only job card data as authorized by you
  • ServiceM8's privacy policy governs their data handling

5. Your Rights (GDPR)

Under GDPR, you have the right to:

  • Access: Request information about data processing (limited due to no storage)
  • Rectification: Correct any stored personal data
  • Erasure: Request deletion of data (minimal data stored)
  • Portability: Export your data (not applicable - no content stored)
  • Objection: Object to data processing
  • Withdrawal: Withdraw consent by disconnecting the add-on

6. Data Security

  • Encryption: All data transmitted via HTTPS/TLS
  • Access Controls: Restricted access to system components
  • Regular Updates: Security patches applied promptly
  • Monitoring: Continuous security monitoring

7. International Data Transfers

  • Translation processing may occur in regions where OpenAI operates
  • Data is processed in accordance with applicable privacy frameworks
  • No long-term storage occurs outside your region

8. Contact Information

Data Controller: Smart Message Translator

Email: help@sparklewindow.com.au

Response Time: Within 72 hours for privacy inquiries

9. Changes to This Policy

We may update this privacy policy to reflect changes in our practices or legal requirements. We will notify users of significant changes through the ServiceM8 add-on interface.

10. Complaints

If you have concerns about how we handle your data, you can:

  • Contact us directly at help@sparklewindow.com.au
  • File a complaint with your local data protection authority
